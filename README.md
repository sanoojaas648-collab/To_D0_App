# To-Do App

A simple and interactive To-Do application built using **HTML, CSS, and JavaScript**.  
This app helps users organize daily tasks, track completed activities, and manage their work efficiently through a clean and responsive interface.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface
- Responsive design
- Easy task management
- Dynamic task updates without page refresh

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
To-Do-App/
│
├── index.html        # Main HTML file
├── style.css         # Styling file
├── script.js         # JavaScript functionality
└── README.md         # Project documentation
```

## ⚙️ How to Run the Project

1. Clone the repository:https://github.com/sanoojaas648-collab/To_D0_App

```bash
git clone <your-repository-link>
```

2. Navigate to the project folder:

```bash
cd To-Do-App
```

3. Open `index.html` in your browser.

OR

Use **Live Server** extension in VS Code for a better development experience.

## 🎯 Usage

- Enter a task in the input field.
- Click the add button to create a new task.
- Click on a task to mark it as completed.
- Delete tasks when they are no longer needed.
